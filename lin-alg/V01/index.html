V1 - Vector Spaces

Example 1

V1 - Vector Spaces (ver. 1)

Let \(V\) be the set of all pairs of numbers \((x_1,y_1)\in\mathbb{R}^2\) together with the operations \begin{align*} (x_1,y_1)\oplus (x_2,y_2)&= (4x_1+x_2, y_1+2y_2)\\ c \odot (x_1,y_1) &= (cx_1,cy_1)\\ \end{align*} Show that scalar multiplication distributes over vector addition, i.e.

\[c\odot \left((x_1,y_1)\oplus(x_2,y_2)\right)=c\odot(x_1,y_1)\oplus c\odot(x_2,y_2).\]

Show that \(V\) nonetheless is not a vector space.

Answer.

\(V\) is not a vector space because vector addition is not associative
