Answer.

\[y= 2 t^{ -2 } + \frac{1}{t^{3}}\]

Find the solution to the given IVP.

\[-\frac{3}{t} +ty'= -2 y \hspace{1em} y( -1 ) = 1\]

Answer.

\[y= 4 t^{ -2 } +3 \frac{1}{t}\]
