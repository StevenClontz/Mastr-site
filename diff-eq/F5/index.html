F5: First order linear IVP

Example 1

F5: First order linear IVP (ver. 1)

Find the solution to the given IVP.

\[-\frac{24}{t} = -t {y'} + 5 \, {y} \hspace{1em} y( -1 ) = 3\]

Answer.

\[{y} = t^{5} - \frac{4}{t}\]
