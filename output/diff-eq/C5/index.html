C5 - Homogeneous second-order linear IVP

Example 1

C5 - Homogeneous second-order linear IVP (ver. 1)

Find the solution to the given IVP.

\[y''-2y'-15y = 0 \hspace{1em} y(0) = 8 , y'(0) = 16\]

Answer.

\[y= 5 \, e^{\left(5 \, t\right)} + 3 \, e^{\left(-3 \, t\right)}\]
